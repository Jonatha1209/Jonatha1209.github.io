## B번 - 자세히 보아야 예쁘다

되게 귀여운 문제입니다.

(A[i]-1의 합) + 1 만큼 보면 최소 한명은 A[i]만큼 봐야합니다.

(A[i]-1의 합)이 n보다 작으면 살고, 아니면 퇴학당하는 문제였습니다.

주의 할 점은, long long int 문을 사용해야 한다는 겁니다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    long long  n,m;
    long long  sum = 0;
    cin >> n >> m;
    for (long long  i=0;i<n;i++){
        long long  x;
        cin >> x;
        sum += (x-1);      
    }
    if (sum < m) {
        cout << "OUT";
    } else {
        cout << "DIMI";
    }

    return 0;
}
```
