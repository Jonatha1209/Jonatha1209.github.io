## 본론

문제는 큐의 대한 기초적인 문제입니다. 대충 알아보자면, 커맨드에 푸시가 들어오면 큐에 cmd[1]출력, pop 이 cmd라면 -1 아니면 왼쪽에 있는에 팝, size 가 cmd라면 큐의 길이, empty가 cmd라면 0 또는 1, front가 cmd라면 -1 아니면 맨앞, back이 cmd라면 que의 맨끝의 있는 요소 또는 -1를 출력하면됩니다. 

코드에 대한 구현이 상당히 귀찮다만, 뭐 어쩌겠습니까 해야죠.

간단한 소스 코드를 첨부했으며, 원리는 다음 블로그를 보시면서 이해하는걸 추천드립니다.

<https://blog.encrypted.gg/934>

```py 

import sys
input =sys.stdin.readline

import collections
que=collections.deque()
for i in range(int(input())):
    cmd=input().split()
    if cmd[0]=="push":
        que.append(cmd[1])
    elif cmd[0]=="pop":
        if len(que):print(que.popleft())
        else:print(-1)
    elif cmd[0]=="size":
        print(len(que))
    elif cmd[0] == 'empty':
        if len(que):
          print(0)
        else:
          print(1)
    elif cmd[0]=="front":
        if len(que):print(que[0])
        else:print(-1)
    elif cmd[0]=="back":
        if len(que):print(que[-1])
        else:print(-1)
```
